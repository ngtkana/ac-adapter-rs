Struct fps::Fps

pub struct Fps<const P: u64> { /* private fields */ }
Expand description

The formal power series (FPS). This object knows a FPS $f$ modulo $x^d$ where $d$ is precision. The precision can be infinite.

Implementation

This is a pair of a vector of coeffs and precision. The infinite precision is represented by usize::MAX.

Invariants

  • coeffs.last() != Some(&Fp::new(0))
  • coeffs.len() <= precision

Examples

use fp2::fp;
type Fps = fp2::Fps<998244353>;
let f = Fps::new(vec![fp!(1), fp!(2)], 4);
let g = f.inv();
let expected = Fps::new(vec![fp!(1), fp!(-2), fp!(4), fp!(-8)], 4);
assert_eq!(g, expected);

Changes the precision. If the new precision is smaller than the current one, the coefficients are truncated.

Examples
use fp2::fp;
type Fps = fp2::Fps<998244353>;
let mut f = Fps::new(vec![fp!(1), fp!(2)], 4);

assert_eq!(f.precision(), 1);
assert_eq!(f.into_coeffs(), vec![fp!(1)]);
let mut f = Fps::new(vec![fp!(1), fp!(2)], 4);
f.set_precision(10);
assert_eq!(f.precision(), 10);
assert_eq!(f.into_coeffs(), vec![fp!(1), fp!(2)]);

pub fn inv(&self) -> Self

Inverse FPS of self.

Requirements

$f_0 \ne 0$

Returns

$f^{-1} \mod x^d$

Complexity

$O(d \log d)$

Examples

type Fps = fp2::Fps<998244353>;
let f = Fps::new(vec![fp!(1), fp!(2)], 4);
let g = f.inv();

source

pub fn sqrt(&self) -> Self

Square root of self.

Requirements

$f_0 = 1$

Returns

$f^{1/2} \mod x^d$

Complexity

$O(d \log d)$

Examples
use fp2::fp;
type Fps = fp2::Fps<998244353>;
let f = Fps::new(vec![fp!(1), fp!(2), fp!(3), fp!(4), fp!(5)], 5);
let g = f.sqrt();
let expected = Fps::new(vec![fp!(1); 5], 5);
assert_eq!(g, expected);
