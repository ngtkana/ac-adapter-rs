重軽分解をします。

使い方

構造体 Hld を使いましょう。

初期化

  • new: 根付き木から構築します。

基本的なメソッド

  • iter_v: パスを分解して、両端点を返すイテレータを作ります。
  • iter_e: それの LCA だけスキップするバージョンです。

便利なショートメソッド

  • dist: 2 点の間の距離を計算します。
  • lca: 2 点の LCA を探します。
  • between: 3 点が一直線上にあるかどうかを判定します。

頂点 ID 翻訳

  • ord: 訪問時刻 → もとの頂点番号
  • time: もとの頂点番号 → 訪問時刻

せっかく計算してるので公開している情報

  • child: 親を消したグラフ
  • size: 部分木のサイズ
  • parent: 受け取った根付き木における親
  • head: heavy path における先頭

Examples

use hld::Hld;
let root = 0;
let n = 4;
let edges = [[0, 1], [0, 2], [2, 3]];
let g = vec![
    vec![1, 2],
    vec![0],
    vec![0, 3],
    vec![2],
];
let hld = Hld::new(root, &g);

assert_eq!(hld.child(), &[vec![2, 1], Vec::new(), vec![3], Vec::new()]);
assert_eq!(hld.time(), &[0, 3, 1, 2]);
assert_eq!(hld.ord(), &[0, 2, 3, 1]);
assert_eq!(hld.parent(), &[0, 0, 0, 2]);
assert_eq!(hld.head(), &[0, 1, 0, 0]);
