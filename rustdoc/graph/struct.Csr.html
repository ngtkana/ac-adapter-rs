Expand description

Compressed Sparse Row format

§Notation

  • $X = [x_0, x_1, \dots, x_{n-1}]$: data
  • $S = [s_0, s_1, \dots, s_{m}]$: boundaries

§Invariants

  • $0 = s_0 \leq s_1 \leq \dots \leq s_m = n$

§Semantics

This struct represents a list of lists $A$ of elements of type T.

$$ A_{i, j} = X[s_i + j] $$

Fields§

§data: Vec<T>

$X = [x_0, x_1, \dots, x_{n-1}]$

§boundary: Vec<usize>
