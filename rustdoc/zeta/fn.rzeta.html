pub fn rzeta<T: Copy>(a: &mut [T], f: impl Fn(T, T) -> T)
Expand description

反転束において、f を加法とするゼータ変換をします。

§制約

  • a の長さが 2 冪
  • f が可換かつ結合的

§効果

a( S ) = f { a ( T ): S ⊆ T }

§計算量

n が配列の長さであるとして、f を丁度 (n lg n) / 2 回呼びます。

§Examples

累積和は add を使うとよいのですが、あえて zeta を使うならばこうです。

let mut a = [1, 2, 4, 8];
rzeta(&mut a, |x, y| x + y);
assert_eq!(a, [15, 10, 12, 8]);
