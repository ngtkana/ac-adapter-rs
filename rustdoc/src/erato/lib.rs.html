1//! Use the two types of sieve of eratosthenes to query.
2//!
3//! # Difference between [`Sieve`] and [`LpdSieve`]
4//!
5//! [`Sieve`] is an ordinary sieve of eratosthenes, which is constructed in O ( n lg lg n ) time,
6//! while [`LpdSieve`] is a table of "least prime divisors'.
7//!
8//! Least prime divisors will accelerate prime factorization, but it takes O ( n lg n ) time to
9//! construct it. Furthermore, it requires a sieve to constructed to the length n + 1, while the
10//! trial division algorithm requires a sieve to constructed to the length √n + 1.
11//!
12//!
13//! # Common usage
14//!
15//! It can be used to check if an integer is a prime number.
16//!
17//! ```
18//! use erato::LpdSieve;
19//! use erato::Sieve;
20//!
21//! let mut sieve = Sieve::new();
22//! assert!(sieve.is_prime(2));
23//! assert!(!sieve.is_prime(20));
24//!
25//! let mut sieve = LpdSieve::new();
26//! assert!(sieve.is_prime(2));
27//! assert!(!sieve.is_prime(20));
28//! ```
29//!
30//!
31//! And it can enumerate all the prime numbers.
32//!
33//! ```
34//! use erato::LpdSieve;
35//! use erato::Sieve;
36//!
37//! let mut sieve = Sieve::new();
38//! let mut prime_numbers = sieve.prime_numbers();
39//! assert_eq!(prime_numbers.next(), Some(2));
40//! assert_eq!(prime_numbers.next(), Some(3));
41//!
42//! let mut sieve = LpdSieve::new();
43//! let mut prime_numbers = sieve.prime_numbers();
44//! assert_eq!(prime_numbers.next(), Some(2));
45//! assert_eq!(prime_numbers.next(), Some(3));
46//! ```
47//!
48//!
49//! # Prime factorization
50//!
51//! `Sieve` provides a trial-division algorithm,
52//!
53//! ```
54//! use erato::Sieve;
55//!
56//! let mut sieve = Sieve::new();
57//! itertools::assert_equal(sieve.prime_factors(84), vec![2, 2, 3, 7]);
58//! ```
59//!
60//! while `LpdSieve` provides a table-lookup algorithm.
61//!
62//! ```
63//! use erato::LpdSieve;
64//!
65//! let mut sieve = LpdSieve::new();
66//! itertools::assert_equal(sieve.prime_factors(84), vec![2, 2, 3, 7]);
67//! ```
68//!
69//! See [`PrimeFactors`] to make unique or run-length encode them.
