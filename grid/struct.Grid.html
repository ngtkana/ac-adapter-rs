pub struct Grid<T> { /* private fields */ }
Expand description

A wrapper to rotate/reflect a grid logically.

Use Dihedral to act this. An affine representation of Dihedral on Map(ℤ², T) is defined by

  • R1 is a 90-degrees rotation, that is (R1 * f)(i, j) = f(w - 1 - j, h).
  • R0S is a transposition, that is (R0S * f)(i, j) = f(j, i).

Thus each the element acts like this:

  • R0: identity.
  • R1: rotate left.
  • R2: anti-position.
  • R3: rotate right.
  • R0S: transposition.
  • R1S: reflection through i = (h - 1) / 2.
  • R2S: anti-transposition.
  • R3S: reflections through j = (w - 1) / 2.
