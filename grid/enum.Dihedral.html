Expand description

An element of a dihedral group D₄.

Formally, D₄ is generated by two element R and S subjecting the following relations,

  • R⁴ = 1,
  • S² = 1,
  • SRS = R³.

Each variant of the enum Dihedral corresponds an element of D₄ as this.

  • R0 : 1
  • R1 : R
  • R2 : R²
  • R3 : R³
  • R0S : S
  • R1S : RS
  • R2S : R²S
  • R3S : R³S
